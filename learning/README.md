# Learning Directory

**Purpose**: Zettelkasten-style learning notes and knowledge capture

---

## Directory Structure

```
learning/
├── README.md (this file)
├── books/           # Book notes and summaries
├── courses/         # Course materials and learnings
├── papers/          # Academic paper notes
└── concepts/        # Atomic concept notes (Zettelkasten)
```

---

## Usage

### Book Notes

Format: `books/[Book Title] - [Author].md`

Example frontmatter:
```yaml
---
tags: [book, topic]
created: YYYY-MM-DD
status: reading|completed
author: Author Name
---
```

### Course Notes

Format: `courses/[Course Name].md`

### Concept Notes

Format: `concepts/[Concept].md`

Atomic notes following Zettelkasten principles:
- One idea per note
- Link to related concepts
- Source references
